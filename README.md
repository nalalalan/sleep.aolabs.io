# sleep.aolabs.io

Single-page PWA for Sleep, an AO Labs low-stimulation digital transition object.

The live experience is intentionally minimal and mostly nonverbal. The in-page paper section carries the explicit research framing so the transition surface does not become a coaching app.

## Local

```powershell
npm run dev
```

Then open `http://127.0.0.1:3051/`.
